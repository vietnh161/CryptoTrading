# CryptoTrading
A project for learning python and react

## Modules
### Home
- Real-time coin price
- Top gainers / losers
### Authentication
- Login/Logout
- 2FA (Google Authenticator)
### Trading Page
- Candlestick chart
- Order Book (Buy / Sell)
- Order Placement Form (Market / Limit)
- Transaction history
### Wallet
- Balance for each coin
- Deposit/Withdrawal
### User Dashboard
- Transaction history
- PnL
- Profile
